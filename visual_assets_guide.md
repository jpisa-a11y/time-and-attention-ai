- **Dimensions:** 1080x1350px (4:5)
- **Layout:** 
  - Background: Solid color (Terracotta, Sage, or Forest Green)

## üé® DESIGN PRINCIPLES

### Organic Authenticity
- **Warmth:** Use terracotta, sage, and warm tones
- **Authenticity:** Show real people, real stories
- **Organic:** Watercolor illustrations, natural textures
- **Purpose-Driven:** Every visual supports the mission

### Accessibility
- **Color Contrast:** Ensure 4.5:1 contrast ratio for text
- **Text Size:** Minimum 14px for body text
- **Alt Text:** Describe all images for screen readers
- **Captions:** Include captions on all videos
- **Diverse Representation:** Show diverse people, abilities, backgrounds

### Consistency
- **Color Palette:** Use brand colors consistently
- **Typography:** Maintain font pairings
- **Spacing:** Use consistent margins and padding
- **Style:** Maintain visual style across platforms
- **Branding:** Include logo and brand elements

### Hierarchy
- **Focal Point:** Clear primary focus
- **Secondary Elements:** Support the main message
- **Readability:** Easy to scan and understand
- **Flow:** Guide viewer's eye naturally
- **Balance:** Symmetrical or intentional asymmetry
